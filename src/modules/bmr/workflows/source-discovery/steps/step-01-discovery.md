# Step 1: Targeted Source Discovery

## YOUR TASK:

Find specific sources to meet identified evidence needs.

## SEQUENCE:

### 1. Identify Need

"{{user_name}}, I'm Morgan. What evidence are you looking for?

**Quick Options:**
[A] Find sources for a specific section of your outline
[B] Find a statistic or data point
[C] Find examples or case studies
[D] Find theoretical framework or methodology sources
[E] Describe what you need

Which applies? (Enter A, B, C, D, or E)"

### 2. Gather Specifics

Based on selection, gather:
- Specific claim or argument needing support
- Type of evidence preferred
- Any known sources to build from
- Quality requirements (peer-reviewed only, etc.)

### 3. Conduct Targeted Search

"**Searching for:** [specific need]

**Search Approach:**
- [Strategy 1]
- [Strategy 2]

**Evaluating sources...**"

### 4. Present Findings

"**Found [N] relevant sources:**

**Recommended:**
1. **[Author (Year)]**: [Title]
   - Relevance: [why this helps]
   - Key finding: [what you can cite]
   - Quality: [assessment]

2. **[Author (Year)]**: [Title]
   - Relevance: [why this helps]
   - Key finding: [what you can cite]
   - Quality: [assessment]

3. **[Author (Year)]**: [Title]
   - Relevance: [why this helps]
   - Key finding: [what you can cite]
   - Quality: [assessment]

**Shall I:**
[A] Add these to your citation library
[B] Find more sources like these
[C] Search for something different
[D] Return to Riley"

### 5. Update Citation Library

Add selected sources to project's citation database.
