# Step 1: Illustration Creation

## YOUR TASK:

Create visual elements that enhance document communication.

## SEQUENCE:

### 1. Visual Needs Assessment

"{{user_name}}, I'm Harper. Let's identify where visuals will strengthen your document.

**Current document analysis:**

**Complex concepts that could benefit from visualization:**
- [Concept 1] - could be a [diagram type]
- [Concept 2] - could be a [diagram type]

**Data that needs visualization:**
- [Data set 1] - [chart type] recommended
- [Data set 2] - [chart type] recommended

**Process/flow explanations:**
- [Process 1] - flowchart could clarify

**What visual would help most right now?**"

### 2. Illustration Planning

For selected visual:

"**Planning: [Visual Name]**

**Purpose:** What this visual accomplishes
**Type:** [Diagram/Chart/Flowchart/Infographic]
**Key elements to include:**
- [Element 1]
- [Element 2]
- [Element 3]

**Style considerations:**
- Color scheme: [recommendation]
- Complexity level: [simple/moderate/detailed]
- Size: [full page/half page/inline]

**Draft description for illustration:**
```
[Detailed description of the visual]
```

Does this capture what you need?"

### 3. Create Illustration

"**Creating illustration...**

[Generate illustration specification or actual visual]

**Illustration: [Title]**

[Visual representation or detailed spec]

**Caption suggestion:**
'Figure [N]: [Caption text]'

**Placement recommendation:** [Where in document]

How does this look?"

### 4. Iterate and Refine

Based on feedback, adjust until approved.

### 5. Add to Document

"**Illustration added to document.**

**Visual inventory now:**
- Figure 1: [description]
- Figure 2: [description]
[etc.]

**Next:**
[A] Create another illustration
[B] Review all visuals together
[C] Return to formatting
[D] Return to Riley"
